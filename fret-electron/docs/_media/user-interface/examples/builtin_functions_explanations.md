Built-In Functions Documentation
--------------------------------

This document describes built-in functions that return the absolute value, maximum, or minimum of real numbers and integers. 

### absReal(x)

Absolute value for reals. The input `x` is a real number. The function returns `x` if `x` is greater than or equal to 0.0 and returns `-x` otherwise.

### absInt(x)

Absolute value for integers. The input `x` is an integer. The function returns `x` if `x` is greater than or equal to 0 and returns `-x` otherwise.

### maxReal(a, b)

Maximum value between two reals. The inputs `a` and `b` are real numbers. The function returns `a` if `a` is greater than or equal to `b` and returns `b` otherwise.

### maxInt(a, b)

Maximum value between two integers. The inputs `a` and `b` are integers. The function returns `a` if `a` is greater than or equal to `b` and returns `b` otherwise.

### minReal(a, b)

Minimum value between two reals. The inputs `a` and `b` are real numbers. The function returns `a` if `a` is less than or equal to `b` and returns `b` otherwise.

### minInt(a, b)

Minimum value between two integers. The inputs `a` and `b` are integers. The function returns `a` if `a` is less than or equal to `b` and returns `b` otherwise.
